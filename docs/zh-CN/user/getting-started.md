# 快速上手

本页给你一条最短可执行路径：从首次启动到建立稳定 SSH 会话。

## 开始前准备

- 可连接的 SSH 目标（`host`、`port`、`username`）。
- 可用认证材料（密码或私钥）。
- 本机到目标主机的网络连通性。

## 5 分钟完成首次会话

1. 从 GitHub Releases 下载并安装 Cosmosh。
2. 打开 Cosmosh，进入服务器创建流程。
3. 新建一个服务器条目，填写主机、端口、用户名和认证方式。
4. 保存后发起 SSH 会话。
5. 如果出现指纹校验提示，核验后再信任。
6. 执行安全命令（如 `pwd` 或 `uname -a`）确认连接成功。

## 首次连接后建议

- 给服务器条目建立统一命名规则（例如 `env-region-role`）。
- 按环境分组主机，减少日常误操作。
- 使用头像菜单“同步设置”前，先在设置中启用“账号同步”。
- 操作完成后显式关闭会话。
