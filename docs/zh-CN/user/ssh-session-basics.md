# SSH 会话基础

## 发起会话

1. 选择目标服务器配置。
2. 发起 SSH 会话。
3. 等待终端提示符就绪。
4. 执行一条无风险验证命令。

## 会话中建议

1. 单个会话尽量保持单一任务上下文。
2. 布局变化后及时调整终端尺寸，避免显示错位。
3. 优先使用明确命令序列，避免长时间漂移式操作。
4. 切换会话前先记录关键命令输出。

## 信任与校验

- 认真核对主机指纹提示。
- 仅信任已验证来源的主机。

## 安全结束会话

1. 确认命令执行完成或已安全取消。
2. 退出远程 shell。
3. 在界面中确认会话已完全关闭。
4. 如需重连，建议发起新的干净会话而非复用旧上下文。

## 常见误区

1. 因主机命名相似而连接到错误环境。
2. 为赶时间跳过主机指纹核验。
3. 任务结束后忘记关闭会话。
