# SSH 会话基础

## 发起会话

1. 选择目标服务器配置。
2. 发起 SSH 会话。
3. 等待终端提示符就绪。
4. 执行一条无风险验证命令。

## 会话中建议

1. 单个会话尽量保持单一任务上下文。
2. 布局变化后及时调整终端尺寸，避免显示错位。
3. 优先使用明确命令序列，避免长时间漂移式操作。
4. 切换会话前先记录关键命令输出。

## Orbit Bar

- 在终端中选中文本后，会在选区附近显示 Orbit Bar。
- 可通过 Orbit Bar 快速执行复制与搜索动作。
- 搜索行为遵循设置中的搜索引擎配置。
- 如不需要 Orbit Bar，可在“设置 > 终端”中关闭。

## 标签页切换

- 按 Ctrl+Tab（macOS 也使用 Control+Tab）打开标签页切换器。
- 持续按住 Ctrl/Control 时，连续按 Tab 可向后轮换目标标签页。
- 按住 Shift 再按 Tab 可反向轮换。
- 切换器打开后，可使用上/下方向键预览目标标签页。
- 松开 Ctrl/Control 后会跳转到当前选中的目标标签页。
- 在切换器中用鼠标点击标签页会立即跳转，即使 Ctrl/Control 仍按住也会生效。

## 信任与校验

- 认真核对主机指纹提示。
- 仅信任已验证来源的主机。

## 安全结束会话

1. 确认命令执行完成或已安全取消。
2. 退出远程 shell。
3. 在界面中确认会话已完全关闭。
4. 如需重连，建议发起新的干净会话而非复用旧上下文。

## 常见误区

1. 因主机命名相似而连接到错误环境。
2. 为赶时间跳过主机指纹核验。
3. 任务结束后忘记关闭会话。
