# 本地终端使用

你可以在 Cosmosh 中直接使用本地终端会话，无需额外打开外部 shell。

## 典型流程

1. 从工作区入口打开本地终端。
2. 执行本地任务命令（构建、检查、脚本）。
3. 远程任务保持在 SSH 会话中执行。
4. 任务结束后关闭本地终端。

## 实践建议

- 避免在单个本地会话中混入过多无关任务。
- 多终端并行时给会话上下文清晰命名或分区。
- 执行高风险命令前先确认路径与目标目录。
- 及时关闭空闲会话，保持工作区聚焦。

## 常见误区

1. 将本应在远程执行的命令误放到本地终端执行。
2. 长时间保留无效终端导致上下文混乱。
3. 在错误工作目录执行破坏性命令。
