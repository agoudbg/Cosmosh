# 故障排查

## 无法创建 SSH 会话

- 重新核对服务器条目的主机、端口与用户名。
- 确认本机到目标主机网络可达。
- 确认目标主机 SSH 服务已启动。

## 主机指纹提示卡住流程

- 通过可信渠道核对指纹值。
- 仅在确认后再信任。
- 如果指纹异常变化，请先停止连接并联系基础设施负责人。

## 认证失败

- 重新核对密码或私钥路径配置。
- 检查私钥格式与权限。
- 确认服务端 SSH 认证策略允许当前方式。

## 终端显示异常

- 调整终端窗口尺寸。
- 重新连接会话。
- 若渲染仍异常，重启应用后再试。

## 会话频繁中断

- 检查 VPN、代理或网络链路稳定性。
- 更换另一台主机对比，确认是否目标主机特有问题。
- 观察中断是否固定时间触发，还是由特定命令触发。

## 快速排查顺序

1. 先确认主机可达与 SSH 服务状态。
2. 再确认认证材料与主机信任状态。
3. 最后用最小命令路径复现实验。

## 截图占位

1. 指纹校验弹窗状态。
2. 认证失败提示状态。
3. 终端会话关闭或错误状态。
